## 6. 隐私边界

公开项目不会接收或展示：

- 原始日报、内部截图、代码、日志和查询语句；
- 公司、产品、人员、团队和真实业务标识；
- 内部域名、主机、地址、端口和本地路径；
- 凭据、账号、角色、活动、渠道或区域的真实标识；
- 无证据支持的调用次数、用户数量、效率百分比或长期生产影响。

允许进入后续公开审核的内容仅限于重新撰写的机制说明、个人贡献、技术决策、验证结果和明确限制。
